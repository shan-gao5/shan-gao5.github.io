Definition 1 (sample space). The set of all possible outcomes of an experiment is called the 𝐬𝐚𝐦𝐩𝐥𝐞 𝐬𝐩𝐚𝐜𝐞\textbf{sample space} and is denoted by Ω\Omega

Example 1. Head or Tails has sample space Ω={H,T}\Omega = \{H, T\},

Definition 2 (Field). A field ℱ\mathcal{F} is a subcollection of the collection of all subsets of Ω\Omega. ℱ⊆{A:A⊆Ω}\begin{aligned} \mathcal{F} \subseteq \{A \colon A \subseteq \Omega\} \end{aligned} This subcollection needs to satisfy:

  1. if A,B∈ℱA, B \in \mathcal{F}, then A∪B∈ℱA \cup B \in \mathcal{F} and A∩B∈ℱA \cap B \in \mathcal{F}

  2. if A∈ℱA \in \mathcal{F} then Ac∈ℱA^c \in \mathcal{F}

  3. the empty set ∅\emptyset belongs to ℱ\mathcal{F}

It follows from the properties of a field ℱ\mathcal{F} that if A1,A2,…,An∈ℱ then ⋃i=1nAi∈ℱ\begin{aligned} \text{ if } A_1,A_2,\ldots, A_n \in \mathcal{F} \text{ then } \bigcup_{i = 1}^{n}A_i \in \mathcal{F} \end{aligned}

We note that ω\omega is notation used for a member of Ω\Omega.

Definition 3 (σ\sigma-field). A σ\sigma-field is a collection of subsets of Ω\Omega that satisfies the following conditions, in addition to those previously defined for fields in general.

  1. ∅∈ℱ\emptyset \in \mathcal{F}

  2. if A1,A2,…∈ℱA_1, A_2, \ldots \in \mathcal{F} then ⋃i=1∞Ai∈ℱ\bigcup_{i=1}^{\infty} A_i \in \mathcal{F}

  3. if A∈ℱA \in \mathcal{F} then Ac∈ℱA^c \in \mathcal{F}

notice that this actually just requires ℱ\mathcal{F} to be closed under countable unions

Example 2. Take any A⊆ΩA \subseteq \Omega, then {∅,A,Ac,Ω}\{\emptyset, A, A^c, \Omega\} is a σ\sigma-field

Example 3. the smallest σ\sigma-field associated with Ω\Omega is the collection {∅,Ω}\{\emptyset, \Omega\}

Example 4. obviously the powerset, {0,1}n\{0,1\}^n is a σ\sigma-field, but the book doesn’t consider this relevant since probabilities cannot be reasonably assigned to all of it’s members.

Recap 1. With any experiment, we may associate a pair (Ω,ℱ)(\Omega, \mathcal{F}), where Ω\Omega is the set of all possible outcomes or elementary events and ℱ\mathcal{F} is a σ\sigma-field of subsets of Ω\Omega which contains all the events in whose occurences we may be interested in.
