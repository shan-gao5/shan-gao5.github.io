\section{median of two sorted arrays}
We need to use binary search, a better name is half space search. 
\begin{codebox}
	\Procname{$\proc{Binary-Search}(A,n,T)$}
	\li $L \gets 0$
	\li $R \gets n-1$
	\li \While $L \leq R$ 
	\li 	\Do $m \gets \lfloor \frac{L+R}{2} \rfloor$
	\li	\If $A[m] < T$
	\li	\Then $L \gets m + 1$
	\li 	\ElseIf $A[m] > T$ 
	\li 	\Then $R \gets m - 1$
	\li 	\ElseNoIf \Return $m$
		\End
	\End


	\end{codebox}
	The principle behind binary search is that we are halving the search space everytime. We can search for the median by searching for the index index to partition the first array in half. A good visualization to draw on paper is to draw both arrays vertically on a histogram. We are in fact searching for the right indices $i$ and $j$, with $j$ being calculated as $\frac{m+n+1}{2} - i$, such that drawing a line between $A[i]$ and $B[j]$, we are able to find $\frac{m+n+1}{2}$ elements under the line, however we've only found the median if drawing a horizontal line crossing through any other element above the line won't intersect the line formed by $A[i]-B[j]$. If so, then the median is given by $\max(A[i],B[j])$.
