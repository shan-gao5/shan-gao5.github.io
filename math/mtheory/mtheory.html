measure theory prerequisites

Definition 1 (algebra). Let \(S\) be a set, an algebra on a set of subsets of \(S\): \(\Sigma_0\) is called an algebra on \(S\) (or an algebra on \(2^S\)) if

  1. \(S \in \Sigma_0\)

  2. \(F \in \Sigma_0 \implies F^c := S\setminus F \in \Sigma_0\)

  3. \(F, G \in \Sigma_0 \implies F \cup G \in \Sigma_0\)

From these rules, obvious other rules follows such as, \(\emptyset \in \Sigma_0\) obviously since \(\emptyset = S^c \in \Sigma_0\). Also letting \(F, G \in \Sigma_0\), then \((F^c \cup G^c)^c = F\cap G \in \Sigma_0\) Thus, an algebra on S is a family of subsets of \(S\) that are stable under finitely many set operations.

This definition is algebraic in the sense that \(\cap\) is the product operation and symmetric difference being \[\begin{aligned} A \Delta B := (A \cup B)\setminus(A \cap B) \end{aligned}\] We can extend this definition to form a \(\sigma\)-algebra on \(S\) which takes the rules and extends them towards their countable definition. That means, if \(\Sigma\) is an algebra on \(S\), such that whenever \(F_n \in \Sigma\) (\(n \in \mathbb{N}\)) then \[\begin{aligned} \bigcup_{n} F_n \in \Sigma \end{aligned}\] Obviously set intersection follows \[\begin{aligned} \bigcap_n F_n = \left( \bigcup_n F_n^c \right)^c \in \Sigma \end{aligned}\]

measurable space

A pair \((S, \Sigma)\), where \(S\) is a set and \(\Sigma\) is a \(\sigma\)-algebra on \(S\), is called a measurable space. An element of \(\Sigma\) is called a \(\Sigma\)-measurable subset of \(S\).

Definition 2 (\(\sigma\)-algebra generated by a class \(\mathcal{C}\) of subsets). Let \(\mathcal{C}\) be a class of subsets of \(S\), then \(\sigma(\mathcal{C})\), the \(\sigma\)-algebra generated by \(\mathcal{C}\) is the smallest \(\sigma\)-algebra \(\Sigma\) on \(S\) such that \(\mathcal{C} \subseteq \Sigma\).

Proposition 1. A simple proposition follows

  • if C is a \(\sigma\)-algebra then \(\sigma(\mathcal{C}) = \mathcal{C}\)

  • \(\sigma(\sigma(\mathcal{C})) = \sigma(\mathcal{C})\)

  • If \(C_1 \subseteq C_2\), then \(\sigma(C_1) \subseteq \sigma(C_2)\)

By construction the smallest sigma algebra is the intersection of all sigma algebras that contain \(\mathcal{C}\). An interesting fact that arises from this is that the intersection of any two sigma algebra remains a sigma algebra.

Example 1. An example would be if \(A, B \subseteq S\), what is \(\sigma(\{A\})\). We just need to add everything that makes it a \(\sigma\)-algebra, that means closure under union and intersection and complements and the set \(S\) itself, that means \(\sigma(\{A\}) = \{ A, A^c = B , \emptyset, S\}\)

Definition 3 (borel \(\sigma\)-algebra). An easy example would be a borel \(\sigma\)-algebra, \(\mathcal{B}(S)\). Indeed, letting \(S\) be a topological space, \(B(S)\) is the borel \(\sigma\)-algebra generated by the family of open subsets of \(S\). \(\mathcal{B}(\mathbb{R})\) is the most important of all \(\sigma\)-algebras, oftenhand shorthanded by \(\mathcal{B}\). \[\begin{aligned} \mathcal{B}(\mathbb{R}) = \sigma(\text{open subsets of $\mathbb{R}$}) \end{aligned}\]

Consider the collection \[\begin{aligned} \pi(\mathbb{R}) := \{ (-\infty, x] \colon x \in \mathbb{R}\} \end{aligned}\] We claim that \[\mathcal{B} = \sigma(\pi(\mathbb{R})\]

Proof.

  • (\(\supset\)) Since they are both \(\sigma\)-algebras, we just need to prove that they contain the same elements. We can do this exhaustively. Taking an element of \(\sigma(\pi(\mathbb{R}))\) as \((-\infty, x)\) for some \(x\), we notice that this is an element of \(\mathcal{B}\), since \((-\infty, x] =\bigcap_{n \in \mathbb{N}} \left(-\infty, x+\frac{1}{n}\right)\)

  • \(\subset\) all that remains to be prove that is every open set \(G\) of \(\mathcal{B}\) is in \(\sigma(\pi(\mathbb{R}))\), we recall from first problem set of analysis 2 that every open set \(G\) is a countable union of open intervals, that means we just need to show that any \((a,b)\) for \(a,b \in \mathbb{R}\) with \(a< b\) is in \(\sigma(\pi(\mathbb{R}))\). But for any \(u\) with \(u > a\), we have that \[\begin{aligned} (a,u] =(-\infty,u] \cap (a,+\infty) = (-\infty,u] \cap (-\infty, a]^c \in \sigma(\pi(\mathbb{R})) \end{aligned}\]

    but for \(\varepsilon = \frac{1}{2}(b-a)\), we get that \[\begin{aligned} (a,b) = \bigcup_n (a,b-\varepsilon \frac{1}{n}] \end{aligned}\] to recover the open interval \((a,b)\) in \(\mathbb{R}\)

Definition 4 (set theoretic limits). "\(A_n \uparrow A\)" means that

  1. it’s Monotonically increasing \(A_n \subseteq A_{n+1}\)

  2. \(\bigcup_n^\infty A_n = A\)

"\(A_n \downarrow A\) means that

  1. Monotonically decreasing \(A_n \supseteq A_{n+1}\)

  2. \(\bigcap_n^\infty A_n = A\)

Definition 5 (d-systems). A collection \(\mathcal{D}\) of subsets of \(S\) is called a d-system if

  1. \(S \in \mathcal{D}\)

  2. Closed under set differences \(A, B \in \mathcal{D}\), then \(A \setminus D \in \mathcal{D}\)

  3. Closed under monotonic limit \((A_n) \subseteq \mathcal{D}\) if \(A_n \uparrow A\) then \(A \in \mathcal{D}\)

Lemma 1. let \(\Sigma\) be a collection of subsets of \(S\), then \[\begin{aligned} \Sigma \text{ is a $\sigma$-algebra } \iff \Sigma \text{ is a d-system and a $\pi$-system } \end{aligned}\] check definition 9 for \(\pi\)-system

Definition 6 (set functions). Let \(S\) be a set, letting \(\Sigma_0\) be an algebra, and let \(\mu_0\) be a non-negative set function

Definition 7 (Additive). \(\mu_0\) is called additive if \(\mu_0 (\emptyset) = 0\) and, for \(F,G \in \Sigma_0\) \[\begin{aligned} F\cap G = \emptyset \implies \mu_0(F \cup G) = \mu_0(F) + \mu_0(G) \end{aligned}\]

It is countable additive if letting \((A_n) \subseteq \Sigma_0\) and \(A_i \cap A_j = \emptyset\) if \(i \neq j\), then \[\mu_0 \left( \bigcup_i A_i \right) = \sum_i \mu_0(A_i)\]

Definition 8 (measure space). letting \(\Sigma_0\) be a \(\sigma\)-algebra on \(S\), from the measurable space \((S,\Sigma_0)\), we can form a measure space if we have a map \(\mu \colon \Sigma_0 \to [0,\infty)\) that is countably additive then the triple \((S, \Sigma_0, \mu)\) is called a measure space

A measure space is finite if \[\mu(S) < \infty\]
\(\sigma\)-finite if \[\exists (S_n \colon n \in \mathbb{N}) \colon \mu(S_n) < \infty (\forall n \in \mathbb{N}) \text{ and } \bigcup S_n = S\]

Finally, we can introduce the probability measure:
A measure \(\mu\) is called a probability measure if \[\mu(S) = 1\] then the measure space \((S, \Sigma, \mu)\) is called a probability triple

An element \(F \in S\) is called \(\mu\)-null if \(\mu(F) = 0\). A statement about of set of points \(s \in S\) is said to hold almost everywhere if, letting the set of points \(s\) be \(F\), so \[F:= \{s \in S: \text{statement} \} \in \Sigma\] almost everwhere if \(\mu(F) = 0\)

Proposition 3 (Continuity 1). Letting \(A_n \in \Sigma\) for all \(n\). If \(A_n \uparrow A\), then \[\begin{aligned} \mu\left(\bigcup_n^\infty A_n\right) = \lim_{n \to \infty} \mu(A_n) \end{aligned}\]

Proof. Letting \(B_1 = A_1\), and setting \(B_n = A_{n+1}\setminus A_n\), notice that all the \(B_i\)’s are disjoint (draw a picture, knowing that \(A_n \subseteq A_{n+1}\) for all \(n\)). We know that, by construction, \[\bigcup_n^\infty B_n = \bigcup_n^\infty A_n\] Then by countable additivity, we have \[\begin{aligned} \mu(\bigcup_i B_i) = \sum_i \mu(B_i) &= \lim_{n \to \infty} \sum_i^n \mu(B_i)\\ &= \lim_{n \to \infty} \mu\left(\bigcup_i^n B_i\right)\\ &= \lim_{n \to \infty} \mu(A_i) \end{aligned}\] ◻

Definition 9 (\(\pi\)-systems). Let \(S\) be a set, a \(\pi\)-system on \(S\) is a family of subsets \(\mathcal{I}\) on \(S\) that is stable under finite intersection, that means letting \(I_1, I_2 \in I\), then \[I_1 \cap I_2 \in \mathcal{I}\]

Now if we generated a \(\sigma\)-algebra on \(\mathcal{I}\), that means \(\Sigma = \sigma(\mathcal{I}))\), and we have two measures \(\mu_1\) and \(\mu_2\) on \((S,\Sigma)\), such that \(\mu_1 = \mu_2\) on \(\mathcal{I}\) and they are both finite, then \(\mu_1 = \mu_2\) on \(\Sigma\). (Recall that generating a \(\sigma\)-algebra means taking the intersection of all \(\sigma\)-algebras that have \(\mathcal{I}\) as a subclass.

Corollary 1. A corollary follows immediately, if two measures agree on a \(\pi\)-system, then they agree on the \(\sigma\)-algebra generated from the \(\pi\)-system.

Theorem 3 (Carathéodory’s Extension Theorem). Let S be a set, let \(\Sigma_0\) be an algebra on \(S\), and let \[\Sigma := \sigma(\Sigma_0)\] Then if \(\mu \colon \Sigma_0 \to [0,+\infty]\) is a countably additive map then there exists a mesure \(\mu\) on \((S, \Sigma)\) such that \[u = \mu_0 \text{ on } \Sigma_0\]

By corollary 1.6.1 that means if \(\mu_0\) is finite, then the induced measure is unique

Definition 10 (Lebesgue measure). Let \(S = (0,1]\). For \(F \subseteq S\), say that \(F \in \Sigma_0\) if \(F\) may be written as a finite union \[(a_1,b_1] \cup \ldots \cup (a_r, b_r]\] where \(r \in \mathbb{N}\), and \(0 \leq a_1 \leq b_1 \leq \ldots \leq a_r \leq b_r \leq\). Then \(\Sigma_0\) is an algebra on \((0,1]\) and \[\Sigma := \sigma(\Sigma_0) = \mathcal{B}(0,1]\] For \(F\) as defined previously, letting \[\mu_0(F) = \Sigma_{k \leq r}(b_k -a_k)\] Then \(\mu_0\) is well defined and additive on \(\Sigma_0\). Moreover \(\mu_0\) is countably additive on \(\Sigma_0\). Hence by the previous theorem, there exists a unique measure \(\mu\) on \((0,1]\) extending \(\mu_0\) on \(\Sigma_0\). The lesbesgue measure on \([0,1]\) is obtained by a trivial modification setting the set \(\{0\}\) to have lebesgue measure 0.

Lemma 4 (Elementary inequalities). Let \((S,\Sigma,\mu)\) be a measured space. Then

  1. \(\mu(A\cup B) \leq \mu(A) + \mu(B) (A,B \in \Sigma)\)

  2. \(\mu\left(\bigcup_{i \leq n} F_i\right) \leq \sum_{i \leq n} \mu(F_i)\)

  3. if \(\mu(S) < \infty\), then \[\mu(A \cup B) = \mu(A) + \mu(B) - \mu(A \cap B) \quad (A,B \in \Sigma)\]

Definition 11 (set theoretical \(\lim \inf\) and \(\lim \sup\)). Let \(\{A_n \colon n \geq 1\} \subseteq \mathcal{F}\) where \(F \subseteq 2^\Omega\), then \[\begin{aligned} \limsup_{n \to \infty} A_n := \bigcap_{n = 1}^\infty \bigcup_{m=n}^\infty A_m = \lim_{n \to \infty} \bigcup_{m=n}^\infty A_m \\ \liminf_{n \to \infty} A_n := \bigcup_{n = 1}^\infty \bigcap_{m=n} A_m = \lim_{n \to \infty} \bigcap_{m=n}^\infty A_m \end{aligned}\]

Proposition 5 (some conclusion on \(\liminf\) and \(\limsup\)). We aim to introduce to elementary properties.

  1. if \(w \in \limsup_{n} A_n\) then \(\forall n \geq 1\) there exists an index \(m_n\) such that \(w \in A_{m_n} \iff \exists \text{a subsequence } \{m_n \colon n \geq 1\}\) such that \(m_n \to \infty\) as \(n \to \infty\) and \(w \in A_{m_n}\) for all \(n \geq 1 \iff\) \(w\) is in infinitely many \(A_n\)’s.

  2. if \(w \in \liminf_{n} A_n\), then there exists an \(n \geq 1\) such that \(w \in A_m\) for all \(m \geq n\)

  3. Obviously, \(\liminf \subseteq \limsup\). HOWEVER, if they are equal, then the set theoretic limit exists and we call it \(\lim_n A_n\), and it’s defined to be equal to the \(\liminf\) and the \(\limsup\). If \(A_n\) is monotonic, then for sure it exists.

Proposition 6 (DE MORGAN’S LAW FOR SET THEORETIC LIMITS). \[\limsup_n A_n^c = (\liminf_n A_n)^c\] and \[\liminf_n A_n^c = (\limsup_n A_n)^c\] and \[(\limsup_n A_n) \setminus (\liminf_n A_n) = \limsup_n (A_n \setminus A_{n+1})\]

Proposition 7 (some more conclusion about set theoretic limits).

  1. \[(\limsup_n A_n) \cap (\limsup_n B_n) \supset \limsup_n (A_n \cap B_n)\] Note that the opposite inclusion is generally not TRUE

  2. \[(\limsup_n A_n) \cup (\limsup_n B_n) = \limsup_n (A_n \cup B_n)\]

  3. \[(\liminf_n A_n) \cap (\liminf_n B_n) = \liminf_n (A_n \cap B_n)\]

  4. \[(\liminf_n A_n) \cup (\liminf_n B_n) \subseteq (\liminf_n (A_n \cup B_n))\]
